# GCP Sovereign Enterprise Platform (DE)

Enterprise-grade, GDPR-compliant GCP Landing Zone and Kubernetes Platform built for regulated workloads in Germany.

## Features

- Secure Landing Zone
- Shared VPC Architecture
- Private GKE Clusters
- IAM Least Privilege
- Centralized Logging
- Budget Enforcement
- SRE Framework
- AI-powered Incident Intelligence
- FinOps Governance

## Regions
Primary: europe-west3 (Frankfurt)

## Deployment

cd environments/dev
terraform init
terraform apply
